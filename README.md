# Реестр договоров и контроль экономии бюджета

Локальная информационная система для ведения реестра договоров, контроля исполнения бюджета, автоматического расчета экономии и формирования аналитических отчетов.

## Версия

1.0 (13.07.2026)

## Особенности

- **Полностью локальная работа** - система работает без подключения к Интернету
- **Без серверов** - не использует серверные технологии
- **Без облачных сервисов** - все данные хранятся локально
- **Без SQL-баз данных** - хранение данных в JSON/CSV файлах
- **Современный веб-интерфейс** - адаптивный дизайн

## Функциональные модули

### Главная панель (Dashboard)
- Обзор ключевых показателей
- Статистика по договорам
- Быстрые действия

### Реестр договоров
- CRUD операции для договоров
- Поиск и фильтрация
- Автоматический расчет экономии
- Экспорт в CSV

### Автоматический расчет экономии
- Экономия = Плановая сумма − Сумма договора
- Перерасход = max(0, Сумма договора − Плановая сумма)
- Процент экономии = Экономия / Плановая сумма × 100%

### Контроль бюджета
- Отслеживание плановых сумм
- Контроль перерасхода
- Анализ по подразделениям

### Аналитика и диаграммы
- Распределение по статусам
- Распределение по подразделениям
- Экономия по месяцам
- Топ поставщиков

### Отчеты
- Общий отчет по договорам
- Отчет по экономии
- Отчет по перерасходу
- Отчет по подразделениям
- Отчет по поставщикам
- Отчет за период

### Справочники
- Управление подразделениями
- Управление поставщиками

### Резервное копирование
- Создание резервных копий
- Восстановление из резервной копии
- Экспорт всех данных

### Журнал действий
- Логирование всех операций
- История изменений

## Установка и запуск

1. Скопируйте папку `Registry` на ваш компьютер
2. Откройте файл `index.html` в любом современном веб-браузере
3. Система готова к использованию

## Требования

- Современный веб-браузер (Chrome, Firefox, Edge, Safari)
- JavaScript должен быть включен

## Использование

### Добавление договора
1. Перейдите в раздел "Реестр договоров"
2. Нажмите кнопку "Добавить договор"
3. Заполните обязательные поля
4. Нажмите "Сохранить"

### Создание отчета
1. Перейдите в раздел "Отчеты"
2. Выберите нужный тип отчета
3. Нажмите "Сформировать"
4. Экспортируйте или распечатайте отчет

### Резервное копирование
1. Перейдите в раздел "Настройки"
2. Нажмите "Создать резервную копию"
3. Файл будет скачан автоматически

## Поля договора

- ID (автоматически)
- Номер договора
- Дата
- Предмет
- Поставщик
- БИН
- Подразделение
- Ответственный
- Плановая сумма
- Сумма договора
- Экономия (автоматически)
- Перерасход (автоматически)
- Статус
- Даты начала и окончания
- Комментарии

## Статусы договоров

- **Активный** - договор в работе
- **Завершен** - договор выполнен
- **Отменен** - договор отменен
- **Ожидает** - договор ожидает выполнения

## Безопасность данных

- Все данные хранятся локально
- Нет передачи данных в Интернет
- Рекомендуется регулярно создавать резервные копии

## Поддержка

При возникновении проблем или вопросов, пожалуйста, обращайтесь к документации или создайте резервную копию перед внесением изменений.
